# Tarea 2
## Para correr:
Para instalar dependencias:
```
npm install
```

Para correr el servidor:
```
npm run dev
```

Extra: si existe problemas para instalar parcel
```
npm install -g parcel-bundler
```

Luego, entrar a <http://localhost:1234/>
